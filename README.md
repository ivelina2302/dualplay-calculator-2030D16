# DualPay Calculator

Уеб приложение за изчисляване на ресто при плащане в лева и евро
по време на прехода към еврото.

## Функционалности
- Въвеждане на дължима сума в евро
- Плащане в евро, лева или и двете
- Автоматично изчисляване на ресто
- Фиксиран курс: 1 евро = 1.95583 лв

## Технологии
- HTML
- JavaScript
